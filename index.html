Project-DevOps — AWS Infrastructure
DevOps · Cloud

Aufbau einer produktionsreifen DevOps-Infrastruktur auf AWS mit EKS, Terraform, Kustomize und vollständigem Monitoring-Stack. Von der Infrastruktur als Code bis zur automatisierten CI/CD-Pipeline — kein manuelles Deployment mehr.

  • VPC, EKS-Cluster, RDS Postgres & Storage vollständig mit Terraform provisioniert
  • Kubernetes-Deployments via Kustomize (Frontend React/Nginx + Backend Node.js)
  • Prometheus + Grafana Monitoring-Stack mit Alert-Regeln
  • CI/CD-Pipeline (GitHub Actions) — Build, Push zu ECR, Deploy auf EKS
  • DevSecOps: AWS IAM, SonarQube, Secrets-Management, RBAC & Network Policies
  • HPA Autoscaling, High-Availability-Konzepte, Chaos-Testing
